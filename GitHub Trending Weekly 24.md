No.1

![](https://githubawesome.com/content/images/2026/02/image-190.png)

SuperCmd is an open-source launcher built with Electron — it has full Raycast extension compatibility. That means existing Raycast extensions just work, no modifications needed. You get hold-to-speak dictation, ElevenLabs text-to-speech, AI actions through OpenAI and Claude, persistent memory across sessions, and Swift native helpers for deep macOS integration. This is Raycast, but open, extensible, and yours.

[https://github.com/SuperCmdLabs/SuperCmd](https://github.com/SuperCmdLabs/SuperCmd?ref=githubawesome.com)

No.2

![](https://githubawesome.com/content/images/2026/02/image-191.png)

Your React app is slow and you have no idea which component is killing it. The Million.js team just dropped React Doctor — a CLI tool that runs a full medical checkup on your codebase. One command. It scans your entire project for performance diseases — unnecessary re-renders, unstable dependency arrays, functions breaking memoization. It doesn't just diagnose, it tells you exactly how to fix it.

[https://github.com/millionco/react-doctor](https://github.com/millionco/react-doctor?ref=githubawesome.com)

No.6

![](https://githubawesome.com/content/images/2026/02/image-195.png)

RTK (Rust Token Killer) stops you from burning money on AI agent test logs. When Claude Code runs npm test and reads 500 lines of "success" logs, you're paying for those tokens. RTK acts as a context window firewall, intercepting terminal commands and stripping noise—turning massive logs into tiny summaries before the AI sees them. Single Rust binary install, runs instantly, cuts API bills by 90% without changing your workflow.

[https://github.com/rtk-ai/rtk](https://github.com/rtk-ai/rtk?ref=githubawesome.com)
