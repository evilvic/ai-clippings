I’ve collected 30 trending self-hosted projects on GitHub — from media automation and monitoring tools to AI chat apps.

![](https://www.youtube.com/watch?v=IS9PcrqLvIQ)

No.1

Your Spotify playlists live in the cloud, but SoulSync makes them yours. It auto-syncs playlists from Spotify, Tidal, and YouTube to your Plex server by downloading missing tracks from Soulseek in FLAC quality. Smart matching identifies what you need, adds album art and synced lyrics from LRClib, organizes files cleanly, and triggers library scans automatically. Includes artist discovery, and a wishlist that retries failed downloads hourly.

[https://github.com/Nezreka/SoulSync](https://github.com/Nezreka/SoulSync?ref=githubawesome.com)

No.2

Running servers but don't want to pay for Datadog or New Relic? Checkmate is open-source uptime and infrastructure monitoring for free. Monitor websites, page speed, SSL certificates, Docker containers, ports, and game servers. Get real-time alerts via email, Discord, Slack, or webhooks, plus public status pages and incident tracking. Stress-tested with 1000+ monitors, it uses just 100MB RAM.

[https://github.com/bluewave-labs/checkmate](https://github.com/bluewave-labs/checkmate?ref=githubawesome.com)

No.3

Running a homelab with Docker containers, and multiple servers but tracking ports in a spreadsheet? portracker auto-discovers every running service, container, and VM across your network—seeing internal container ports versus published host ports. Add other portracker instances as peers and view your entire infrastructure from one dashboard with hierarchical grouping.

[https://github.com/mostafa-wahied/portracker](https://github.com/mostafa-wahied/portracker?ref=githubawesome.com)

No.4

Backlogia is a self-hosted game library aggregator that pulls everything into one dashboard—Steam, Amazon, EA, Xbox Game Pass, and local folders. Smart deduplication shows games bought on multiple platforms once with all purchase info. Enriched with IGDB metadata—ratings, release dates, genres. Filter by platform, sort by rating or playtime.

[https://github.com/sam1am/backlogia](https://github.com/sam1am/backlogia?ref=githubawesome.com)

No.5

Need a human decision mid-workflow but don't want to build an entire approval system? Ask4Me gives you Human-in-the-Loop in one synchronous request. Send an API call with buttons or text input, get a notification link on your phone via ServerChan or Apprise, click a button or type text, and the original HTTP long-poll request receives the result immediately.

[https://github.com/easychen/ask4me](https://github.com/easychen/ask4me?ref=githubawesome.com)

No.6

Outline is an open-source knowledge base with real-time collaborative editing, a Notion-style interface, slash commands, nested documents, collections, permissions, and version history. Markdown compatible with import/export. But you host it—your infrastructure, your data. Docker Compose with PostgreSQL, Redis, and S3 storage.

[https://github.com/outline/outline](https://github.com/outline/outline?ref=githubawesome.com)

No.7

You run a Plex server, but your friend's login gets shared with their roommate, then their cousin—now you're streaming to twelve people across three continents. Tracearr is a detection system that catches account sharers automatically. Five rule types: impossible travel, simultaneous locations, device velocity, concurrent stream limits, and geo restrictions. Discord webhooks fire instantly when rules trigger.

[https://github.com/connorgallopo/Tracearr](https://github.com/connorgallopo/Tracearr?ref=githubawesome.com)

No.8

HomeDockOS is a complete cloud operating system running on anything—Raspberry Pi Zero to workstations, Linux to macOS to Windows, ARM to x86. The killer feature is Prism Window Manager, bringing an actual desktop experience to your browser with real multitasking, multiple themes, and simultaneous tasks. One-click app store, pre-configured SSL, and automatic migration from Umbrel or CasaOS.

[https://github.com/BansheeTech/HomeDockOS](https://github.com/BansheeTech/HomeDockOS?ref=githubawesome.com)

No.9

Calibre-Web Automated combines Calibre's features with Calibre-Web's lightweight interface. Drop ebooks into an ingest folder for automatic analysis, conversion, and import.Supports 27 formats with conversion to EPUB, MOBI, AZW3, KEPUB, or PDF. UI changes to covers and metadata apply to the actual ebook files, so edits show up on devices like Kindle.

[https://github.com/crocodilestick/Calibre-Web-Automated](https://github.com/crocodilestick/Calibre-Web-Automated?ref=githubawesome.com)

No.10

FrankMD is a self-hosted markdown editor for blog writing.Rails app with no database—works directly with markdown files on your filesystem. Includes AI helpers for S3 image upload, Nano Banana image generation, and YouTube embed search.CodeMirror syntax highlighting, live preview, and visual table editor. Offline local storage with recovery dialog showing side-by-side diffs after crashes.

[https://github.com/akitaonrails/FrankMD](https://github.com/akitaonrails/FrankMD?ref=githubawesome.com)

No.11

Invoice Builder is an offline-first invoicing app for freelancers.Desktop app with no cloud dependency. Create invoices and quotations, export PDFs, configure number and date formatting. Your data stays in a local database file.Choose SQLite for single-user or PostgreSQL for multi-device by connecting your own server. Optional Docker deployment for self-hosting.Built with React and Node.js.

[https://github.com/piratuks/invoice-builder](https://github.com/piratuks/invoice-builder?ref=githubawesome.com)

No.12

Octelium is a self-hosted unified secure access platform on Kubernetes. Functions as remote access VPN, ZTNA platform, API gateway, AI gateway, MCP gateway, PaaS, and ngrok alternative—all in one. WireGuard and QUIC tunnels for clients, clientless BeyondCorp mode for browsers. Layer-seven per-request access control using CEL and Open Policy Agent—no blanket network rules.

[https://github.com/octelium/octelium](https://github.com/octelium/octelium?ref=githubawesome.com)

No.13

Kino is a terminal client for Plex or Jellyfin with keyboard-driven, Vim-style navigation. Use hjkl to move, f for global search, slash for local filtering, space for playlists, i for detailed metadata inspector. Auto-detects your server type and walks through authentication on first launch. Finds and configures mpv, VLC, IINA, or Celluloid with automatic resume support. Sort by title, date, or rating with one keystroke.

[https://github.com/mmcdole/kino](https://github.com/mmcdole/kino?ref=githubawesome.com)

No.14

OpenWorkflow provides durable workflows using your existing database without requiring an orchestration cluster like Temporal.Each operation wraps in a memoized step. If the server crashes after a step completes (like charging a card), that step won't re-execute on resume. Workflows resume via deterministic replay.Sleep for seconds or months without resource overhead. Horizontal scaling by adding workers.

[https://github.com/openworkflowdev/openworkflow](https://github.com/openworkflowdev/openworkflow?ref=githubawesome.com)

No.15

PriceGhost is a self-hosted price tracker that runs four extraction methods in parallel—JSON-LD, meta tags, CSS selectors, and AI. Each method votes on the price. When they agree, you're set. When they disagree, you see all candidates with context and make the final call. AI verification catches mistakes like scraping the savings amount instead of the actual price. Works without AI using multiple scraping strategies.

[https://github.com/clucraft/PriceGhost](https://github.com/clucraft/PriceGhost?ref=githubawesome.com)

No.16

Company uses AWS S3 but sharing files with clients means downloading, emailing, and credential risks? Safebucket gives you secure file sharing on existing infrastructure. Open-source platform creates buckets with role-based access—on any S3-compatible storage. Works with AWS S3, Google Cloud Storage, MinIO, whatever you use. Plug in corporate SSO to eliminate local logins—users share files with existing identities.

[https://github.com/safebucket/safebucket](https://github.com/safebucket/safebucket?ref=githubawesome.com)

No.17

Shelfmark is a web interface that aggregates book and audiobook in one place with a real-time download queue.Features include template-based file naming, separate configs for books and audiobooks, hardlinks and custom directories for Audiobookshelf integration, multi-user support with admin management, per-user download destinations, and four auth methods including OIDC and proxy auth.

[https://github.com/calibrain/shelfmark](https://github.com/calibrain/shelfmark?ref=githubawesome.com)

No.18

hash-calendar is a client-side calendar app that stores all events, settings, and timezones in a compressed URL hash. Share the URL to give someone the full calendar with no backend sync required. Optional password protection uses AES-GCM encryption without server involvement. Includes multiple views, world clock with saved timezones. QR codes for mobile handoff, and browser notifications.

[https://github.com/supunlakmal/hash-calendar](https://github.com/supunlakmal/hash-calendar?ref=githubawesome.com)

No.19

homescreen-hero is a self-hosted dashboard for Plex servers with drag-and-drop widgets. Combines Tautulli streaming metrics, Jellyseerr request management, and Plex library stats in one interface. Automated collection rotation schedules collections to swap on your homescreen using Weighted or LRU strategies. Library cleanup tools fix broken "Date Added" timestamps, mark shows unwatched to reset Continue Watching.

[https://github.com/trentferguson/homescreen-hero](https://github.com/trentferguson/homescreen-hero?ref=githubawesome.com)

No.20

Doppelganger is a self-hosted browser automation platform built on Playwright with a drag-and-drop block editor for clicks, typing, waits, hovers, and JavaScript execution. Proxy rotation with HTTP and SOCKS support, automatic IP switching per task, and one-click proxy list imports. Auto-captures screenshots and recordings, stores cookies, with a captures tab for viewing or downloading assets.

[https://github.com/mnemosyne-artificial-intelligence/doppelganger](https://github.com/mnemosyne-artificial-intelligence/doppelganger?ref=githubawesome.com)

No.21

DDWrap is a lightweight Qt wrapper around dd written in Python. Select source image and target device graphically with real-time progress showing percentage and estimated time remaining. Final confirmation dialog shows device capacity, partition layout, and SMART health data before writing. Displays the actual dd command being executed.

[https://github.com/HarderLemonade/ddwrap](https://github.com/HarderLemonade/ddwrap?ref=githubawesome.com)

No.22

selfhost-roi-calculator is a single HTML file that calculates whether self-hosting saves money compared to streaming services. Input NAS and hard drive costs, monthly power usage in watts × kWh price plus VPS/VPN/Usenet costs, and streaming subscriptions you'd cancel. Shows monthly difference and ROI payback time in months or years. Add custom services as needed.

[https://github.com/value1338/selfhost-roi-calculator](https://github.com/value1338/selfhost-roi-calculator?ref=githubawesome.com)

No.23

nutalert is a self-hosted monitoring system for Network UPS Tools. Connects to your NUT server, auto-detects UPS devices on first launch, and generates default config automatically. Real-time visualization of battery level, load, runtime, and input voltage on port 8087. Customizable alerts trigger on specific conditions with push notifications to Discord, Slack, Email, Telegram, or 80+ other services via Apprise.

[https://github.com/rmfatemi/nutalert](https://github.com/rmfatemi/nutalert?ref=githubawesome.com)

No.24

FaveSwitch is a self-hosted manager for Plex, Jellyfin, Emby, Stremio, and Audiobookshelf. Switch between any user on your servers from one interface and add or remove favorites instantly. Stats dashboard shows total favorite counts by user and media type. Unified search across all integrations with cache warming for fast suggestions. Works with admin API keys, runs in one Docker container on port 5050.

[https://github.com/ponzischeme89/FaveSwitch](https://github.com/ponzischeme89/FaveSwitch?ref=githubawesome.com)

No.25

Veritas Kanban is a lightweight kanban board where tasks are plain markdown files. Full REST API for agent integration with built-in agent service using OpenClaw orchestration. Click "Start Agent" on a code task—the server writes to an agent queue, the agent picks up work, updates status, tracks time, commits code, and calls the completion endpoint when done.

[https://github.com/BradGroux/veritas-kanban](https://github.com/BradGroux/veritas-kanban?ref=githubawesome.com)

No.26

pam is a CLI tool for saving and executing SQL queries. Built with Bubble Tea TUI, it stores queries with tags and descriptions and executes them across PostgreSQL, MySQL, SQLite, SQL Server, or Oracle. File-based storage—no external database. Queries stored as files you can grep. Cross-platform single Go binary. Install with go install or use the Nix flake for NixOS.

[https://github.com/eduardofuncao/pam](https://github.com/eduardofuncao/pam?ref=githubawesome.com)

No.27

DVinyl is a self-hosted collection manager for vinyl, CDs, and cassettes with Discogs API integration. Add items by scanning barcodes or entering release IDs to get metadata and market estimates. Unified library for all formats, wishlist system, and authentication for sharing your collection with friends. Localized in English and French with mobile-optimized dark and light modes.

[https://github.com/Kyonew/DVinyl](https://github.com/Kyonew/DVinyl?ref=githubawesome.com)

No.28

NeatMail is an AI-powered email organizer that watches your Gmail in real-time via Pub/Sub webhooks, classifies emails with GPT-4 as they arrive, and applies labels in your Gmail interface. Preset categories include Action Needed, Pending Response, Automated Alerts, Discussion, and Marketing. You can create custom labels with custom colors. 95% confidence threshold—only labels when the AI is confident.

[https://github.com/Lakshay1509/NeatMail](https://github.com/Lakshay1509/NeatMail?ref=githubawesome.com)

No.29

xPrivo is an open-source AI chat app that stores conversations in browser localStorage with zero logging and no account required. Self-host or use the hosted version at xprivo.com with EU-hosted Mistral 3 and DeepSeek V3.2 models that guarantee zero data retention. Bring your own API keys from OpenAI or Anthropic to use their models directly, or connect Ollama for offline operation.

[https://github.com/xprivo/ai-chat](https://github.com/xprivo/ai-chat?ref=githubawesome.com)

No.30

CronPulse is a self-hosted monitoring service for scheduled jobs. Jobs ping start and OK endpoints—if the expected OK doesn't arrive, you get instant alerts via webhook to Slack, Discord, PagerDuty, or optional email. SQLite database with volume persistence, full REST API with documentation, and web UI on port 8000. Enterprise security features include JWT auth, HTTPS via reverse proxy, and CORS restriction.

[https://github.com/techfort/cronpulse-community](https://github.com/techfort/cronpulse-community?ref=githubawesome.com)
