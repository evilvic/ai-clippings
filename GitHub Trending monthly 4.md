Today we're look at 33 popular projects from last month. If you've been following the channel, you've probably seen most of these in our previous episodes. This is our fourth monthly recap, bringing them all together so you can see what the developer community was obsessing over in January.

![](https://www.youtube.com/watch?v=FAzQa4C0AYk)

No.1

![](https://githubawesome.com/content/images/2026/02/image-33.png)

Created by an Anthropic hackathon winner, this repository is a battle-tested configuration suite that turns Claude Code from a chatbot into a senior engineer. Nearly 40,000 star. Includes custom "Skills" for test-driven development and automated "Hooks" that lint and type-check every file the AI writes.Stop fighting with context windows. Start shipping production code.

[https://github.com/affaan-m/everything-claude-code](https://github.com/affaan-m/everything-claude-code?ref=githubawesome.com)

No.2

![](https://githubawesome.com/content/images/2026/02/image-34.png)

Want to work like the AI startup Meta bought for $2 billion? Planning with Files is a Claude Code skill that brings the workflow behind Manus straight into your terminal. The secret: Markdown as working memory. Instead of cramming everything into the AI's fragile context, you save notes, progress, and checkpoints on disk. Each file becomes a scratchpad, letting Claude remember what matters across tasks.

[https://github.com/OthmanAdi/planning-with-files](https://github.com/OthmanAdi/planning-with-files?ref=githubawesome.com)

No.3

![](https://githubawesome.com/content/images/2026/02/image-35.png)

Agent Browser is a CLI from Vercel Labs designed for AI agents. Standard headless browsers waste thousands of tokens sending raw, messy DOM trees that confuse your model. Agent Browser strips the web down to essentials, reducing context usage by 93% while still letting your agent click, type, and navigate. Combines a Rust binary with a Playwright daemon. Gives your AI a clean way to surf the web without blowing your budget.

[https://github.com/vercel-labs/agent-browser](https://github.com/vercel-labs/agent-browser?ref=githubawesome.com)

No.4

![](https://githubawesome.com/content/images/2026/02/image-36.png)

DeepTutor is an AI study tool that builds a knowledge graph from your course materials. You can ask questions and get step-by-step explanations with citations showing the source. It generates practice questions, connects concepts across papers, and creates visual explanations. A notebook system tracks your learning, and it can suggest research ideas based on what you've studied.

[https://github.com/HKUDS/DeepTutor](https://github.com/HKUDS/DeepTutor?ref=githubawesome.com)

No.5

![](https://githubawesome.com/content/images/2026/02/image-37.png)

You're sitting there with fifty different document types scattered everywhere—PDFs that won't cooperate, random Excel files, scanned receipts, emails—and you're supposed to make sense of all this? Kreuzberg handles fifty-seven file formats. Documents, spreadsheets, images, emails. OCR works on scanned docs, pulls tables out cleanly, cracks password-protected PDFs. Runs locally—no GPU, no cloud upload requirement.

[https://github.com/kreuzberg-dev/kreuzberg](https://github.com/kreuzberg-dev/kreuzberg?ref=githubawesome.com)

No.6

![](https://githubawesome.com/content/images/2026/02/image-38.png)

Vercel Labs released json-render, a library for AI-generated UI that constrains what the model can output. You provide a catalog of allowed components (Card, Metric, Button, etc.), and the AI can only generate JSON using those components.Users can prompt "make me a revenue dashboard" and get UI generated in real-time with streaming output.

[https://github.com/vercel-labs/json-render](https://github.com/vercel-labs/json-render?ref=githubawesome.com)

No.7

![](https://githubawesome.com/content/images/2026/02/image-39.png)

Ralph is a bash script that runs AI coding sessions in a loop until a feature is complete. You provide a PRD as JSON, and it spawns Claude or Amp instances to work through user stories one at a time.Each iteration starts with clean context—only git commits, a progress file, and completed story markers carry over. It runs typechecks and tests, commits if they pass, updates an AGENTS.md file with notes for the next iteration, then restarts for the next story.

[https://github.com/snarktank/ralph](https://github.com/snarktank/ralph?ref=githubawesome.com)

No.8

![](https://githubawesome.com/content/images/2026/02/image-40.png)

MapToPoster is a Python script that generates city map posters from OpenStreetMap data. Run a command with a city name and it geocodes the location, pulls street networks, parks, and waterways, then renders a 300 DPI print-ready poster. It includes 17 themes like "neon cyberpunk," "Japanese ink," and "blueprint." You can create custom themes by editing a JSON file with colors and road hierarchy settings.

[https://github.com/originalankur/maptoposter](https://github.com/originalankur/maptoposter?ref=githubawesome.com)

No.9

![](https://githubawesome.com/content/images/2026/02/image-41.png)

OpenWork is an open-source desktop app that provides a UI for OpenCode, an autonomous coding agent. You select a workspace, start a run, watch the execution plan update, and approve permissions when requested. It can run locally or connect to a remote OpenCode server. A skill manager lets you install and share workflows. There's also Owpenbot, a WhatsApp bot version for running workflows via text.

[https://github.com/different-ai/openwork](https://github.com/different-ai/openwork?ref=githubawesome.com)

No.10

![](https://githubawesome.com/content/images/2026/02/image-42.png)

GitHub released SDKs that let you embed Copilot in your own applications. The way it works: your app calls the SDK, which spawns and manages a Copilot CLI process in the background. Communication happens over JSON-RPC. The CLI is where the actual AI inference happens—your code just sends requests and gets responses back.Useful if you're building developer tools and want Copilot functionality without reinventing it.

[https://github.com/github/copilot-sdk](https://github.com/github/copilot-sdk?ref=githubawesome.com)

No.11

![](https://githubawesome.com/content/images/2026/02/image-43.png)

mapcn — add stunning maps to your web apps with just one command setup. Built on MapLibre GL, mapcn gives you full access to mapping features right out of the box. it’s all ready to go, with sensible defaults that work perfectly for any project. it’s theme-aware, so it automatically adapts to light and dark modes. it lets you add markers and tooltips, plus draw routes and paths on your maps.

[https://github.com/AnmolSaini16/mapcn](https://github.com/AnmolSaini16/mapcn?ref=githubawesome.com)

No.12

![](https://githubawesome.com/content/images/2026/02/image-44.png)

If you're still paying for ElevenLabs, you need to see this. Qwen dropped Qwen3-TTS, and it's fully open-source. It does 3-second voice cloning, but the feature is "Voice Design." You type a prompt like "a nervous teenager trying to hide a secret," and it generates that exact performance. It's built on a new Dual-Track architecture that hits 97-millisecond latency—you can use this for real-time voice agents.

[https://github.com/QwenLM/Qwen3-TTS](https://github.com/QwenLM/Qwen3-TTS?ref=githubawesome.com)

No.13

![](https://githubawesome.com/content/images/2026/02/image-45.png)

Stop staring at your terminal waiting for Claude to finish. 1Code is an open-source GUI that makes AI coding parallel. You give Claude Code a task and your terminal is locked for ten minutes. 1Code fixes this with a visual dashboard that lets you run multiple tasks at once. One agent fixes a bug while another writes docs—each in its own isolated Git worktree.

[https://github.com/21st-dev/1code](https://github.com/21st-dev/1code?ref=githubawesome.com)

No.14

![](https://githubawesome.com/content/images/2026/02/image-46.png)

Antigravity Awesome Skills is a library of 631+ Skills that teach AI coding assistants how to do things. Each skill is a set of instructions for a specific dev task: setting up Terraform infrastructure, profiling performance bottlenecks, wiring up AWS automation, following mobile development patterns, building voice AI pipelines. Like giving your agent access to a senior dev's playbook for each problem domain.

[https://github.com/sickn33/antigravity-awesome-skills](https://github.com/sickn33/antigravity-awesome-skills?ref=githubawesome.com)

No.15

![](https://githubawesome.com/content/images/2026/02/image-47.png)

This might be the end of buying massive GPU clusters just for memory. Engram is DeepSeek's new architecture that separates "thinking" from "remembering." Right now, AI models waste compute just to recall static facts. Engram adds a dedicated memory module using modernized N-gram lookups. Gives the AI instant, O(1) access to knowledge without the heavy processing.

[https://github.com/deepseek-ai/Engram](https://github.com/deepseek-ai/Engram?ref=githubawesome.com)

No.16

![](https://githubawesome.com/content/images/2026/02/image-48.png)

Cloudflare dropped this repo: Moltworker. It lets you Run OpenClaw personal AI assistant in a Cloudflare Sandbox. No more renting a VPS, no more exposing open ports to hackers. It uses Cloudflare Workers for logic, R2 for memory, and Zero Trust for security. Works with Telegram, Discord, Slack—basically every chat platform. Cold starts take 1-2 minutes but then it stays alive indefinitely on the free tier.

[https://github.com/cloudflare/moltworker](https://github.com/cloudflare/moltworker?ref=githubawesome.com)

No.17

![](https://githubawesome.com/content/images/2026/02/image-49.png)

Awesome OpenClaw Skills is a curated, community-driven list of ready-to-use skills, plugins, and tools. It's a plugin marketplace meets cheat sheet for OpenClaw: task-oriented skills, integrations, helpers and utilities, examples you can drop into your agent workflows today. This repo gives you one place to browse, compare, and install. Takes your AI agents from barebones to battle-ready without endless setup.

[https://github.com/VoltAgent/awesome-openclaw-skills](https://github.com/VoltAgent/awesome-openclaw-skills?ref=githubawesome.com)

No.18

![](https://githubawesome.com/content/images/2026/02/image-50.png)

Craft just fixed what everyone hates about Mermaid diagrams. Standard Mermaid rendering looks kind of dated, and theming it means wrestling with CSS classes. beautiful-mermaid gives you stunning SVGs or ASCII art from the same source, zero DOM required. Ships with 15 curated themes plus full Shiki compatibility so you can use any VS Code theme. Supports flowcharts, sequence diagrams, state machines, class diagrams, ER diagrams.

[https://github.com/lukilabs/beautiful-mermaid](https://github.com/lukilabs/beautiful-mermaid?ref=githubawesome.com)

No.19

![](https://githubawesome.com/content/images/2026/02/image-51.png)

If you use Homebrew every day and hate how slow it feels, check out ZeroBrew. It's an open-source Rust replacement for Homebrew that's way faster—5× faster on first installs and up to 20× faster on repeats. It uses a content-addressable cache and parallel downloads, and pulls from the same Homebrew bottles you already use. The best part? It's drop-in compatible. You can try it without wrecking your current brew setup.

[https://github.com/lucasgelfond/zerobrew](https://github.com/lucasgelfond/zerobrew?ref=githubawesome.com)

No.20

![](https://githubawesome.com/content/images/2026/02/image-52.png)

Hive is a self-evolving agent framework. In standard frameworks, when an agent fails, it crashes. In Hive, failures trigger an evolution loop. The system captures the error, spins up a coding agent to rewrite the graph logic, and redeploys a smarter version automatically. Uses natural selection to debug your code. Has built-in budget controls and human-in-the-loop approval, so it can survive in production without you babysitting the logs.

[https://github.com/adenhq/hive](https://github.com/adenhq/hive?ref=githubawesome.com)

No.21

![](https://githubawesome.com/content/images/2026/02/image-53.png)

Nanobot is a personal AI assistant in ~4,000 lines of Python. Built as a simpler alternative to OpenClaw (430,000 lines) — same core features (persistent memory, web search, background agents), way less code. Runs on Telegram or WhatsApp. The codebase is small enough to read through and understand, so you can customize how memory works, add skills, or swap components without reverse-engineering a massive framework.

[https://github.com/HKUDS/nanobot](https://github.com/HKUDS/nanobot?ref=githubawesome.com)

No.22

![](https://githubawesome.com/content/images/2026/02/image-54.png)

It identifies and removes AI patterns, then injects real personality into your writing. It scans for telltale AI marks—overused phrases, flat structures, marketing jargon—and rewrites your sentences so they feel authentic.

[https://github.com/blader/humanizer](https://github.com/blader/humanizer?ref=githubawesome.com)

No.23

![](https://githubawesome.com/content/images/2026/02/image-55.png)

if you love the speed of C but hate the lack of features, you need to see this. Meet Zen-C — a library that finally gives C the "batteries-included" experience. We all know C doesn't have built-in hash maps, dynamic vectors, or safe strings, right? Zen-C adds all of that. It’s a lightweight toolkit that handles networking, and data structures with a clean, unified API. It basically lets you write code that feels as Python, while keeping the performance of C.

[https://github.com/z-libs/Zen-C](https://github.com/z-libs/Zen-C?ref=githubawesome.com)

It even shows todo progress, so you always know what’s done and what’s still running.

Built on the Agno framework, it's a data analyst that never clocks out.

No.30

![](https://githubawesome.com/content/images/2026/02/image-62.png)

DeepSeek dropped OCR-2. Instead of processing images the standard way, they built something called Visual Causal Flow that mimics how humans read documents. Handles dynamic resolution so it can process PDFs fast—they're claiming parity with their first OCR model but with better accuracy. Works with both vLLM for production speed and regular Transformers. Supports everything from clean Markdown conversion to layout-free OCR when documents are messy.

No.31

![](https://githubawesome.com/content/images/2026/02/image-63.png)

Managing AWS from the console is powerful… but it’s also overwhelming. taws brings AWS into your terminal — as a fast, keyboard-driven UI. It lets you browse 94 AWS resource types across 60 services, switch profiles and regions instantly, and see changes in real time. Everything is Vim-style. Navigate, filter, and inspect resources without touching a mouse. You can start, stop, or terminate EC2 instances directly from the UI.

[https://github.com/huseyinbabal/taws](https://github.com/huseyinbabal/taws?ref=githubawesome.com)

No.33

![](https://githubawesome.com/content/images/2026/02/image-65.png)

AI coding assistants forget your preferences between sessions. You explain "we use Zustand not Redux" or "always run the linter," and the next day it's gone. Trellis fixes this with hooks that auto-inject your coding standards into every conversation. You tell it once to use Zustand, and it saves that to a spec file that loads automatically in future sessions. It updates its own specs based on what it learns from each task.

[https://github.com/mindfold-ai/Trellis](https://github.com/mindfold-ai/Trellis?ref=githubawesome.com)
