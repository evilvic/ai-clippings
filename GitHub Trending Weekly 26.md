No.1

![](https://githubawesome.com/content/images/2026/03/image-68.png)

Want AI code review without handing your proprietary code to a third-party SaaS? OpenReview from Vercel Labs is the answer. Deploy it to Vercel, connect it as a GitHub App, and just mention @openreview in any pull request. It spins up a sandboxed environment, clones your branch, and deploys a Claude-powered agent to review the diff. If it finds lint errors or formatting issues, it automatically commits and pushes the fixes directly to your PR branch.

[https://github.com/vercel-labs/openreview](https://github.com/vercel-labs/openreview?ref=githubawesome.com)

No.3

![](https://githubawesome.com/content/images/2026/03/image-70.png)

SSD stands for 'Speculative Speculative Decoding'. Standard speculative decoding is fast, but it forces the fast model to wait for the slow model to verify tokens. SSD breaks that barrier by parallelizing the process across distinct hardware—literally predicting the verification outcomes before they even happen! The result? A lossless inference engine that runs up to 2x faster than the strongest baselines in the world.

[https://github.com/tanishqkumar/ssd](https://github.com/tanishqkumar/ssd?ref=githubawesome.com)

![](https://githubawesome.com/content/images/2026/03/image-71.png)

If you use Claude Code, you know the terminal tells you almost nothing about what the AI is actually doing under the hood. claude-devtools fixes this. It visualizes your entire hidden context window: every file read, every tool call, recursive subagent trees, and exactly where your tokens are going.

[https://github.com/matt1398/claude-devtools](https://github.com/matt1398/claude-devtools?ref=githubawesome.com)

No.28

![](https://githubawesome.com/content/images/2026/03/image-95.png)

Fractals is a recursive agentic task orchestrator. Give it a high-level goal and it dynamically grows a self-similar tree of subtasks, running each leaf in an isolated Git worktree using a swarm of AI agents in parallel. Once the leaves are done, a merger agent combines branches bottom-up — resolving conflicts until a single unified project emerges at the root. Incredibly clever architecture for massive AI workflows.

[https://github.com/TinyAGI/fractals](https://github.com/TinyAGI/fractals?ref=githubawesome.com)
