This is GitHub Trending Weekly, Episode 28. In this video, I’ve handpicked 33 trending open-source projects on GitHub.

![](https://www.youtube.com/watch?v=TpWlE0HCZwM)

No.1

![](https://githubawesome.com/content/images/2026/03/image-406.png)

Project-NOMAD stands for Node for Offline Media, Archives, and Data, and it's exactly what that sounds like. It's a self-contained, Docker-orchestrated web server designed to run completely without internet access. Through a single Command Center UI, you can pre-download full Wikipedia archives, detailed offline maps, survival and medical PDFs, and spin up a local AI assistant powered by Ollama.

[https://github.com/Crosstalk-Solutions/project-nomad](https://github.com/Crosstalk-Solutions/project-nomad?ref=githubawesome.com)

No.2

![](https://githubawesome.com/content/images/2026/03/image-407.png)

AI agents write frontend code without any way to verify it actually works in a browser. Expect closes that loop. It's a CLI skill for Claude Code that automatically scans unstaged git changes or branch diffs, generates a test plan, and executes it against a real live browser. When your agent finishes building a feature, it tests the UI itself before opening a pull request. Your agent writes the code, checks that it works, then asks for review.

[https://github.com/millionco/expect](https://github.com/millionco/expect?ref=githubawesome.com)

No.3

![](https://githubawesome.com/content/images/2026/03/image-408.png)

Every Claude Code session starts from scratch. Same mistakes, same token burn, no memory of what worked last time. OpenSpace runs alongside your agent and watches it work. When it successfully solves something complex, a weird TypeScript error or a resilient data service, OpenSpace extracts that logic and saves it as a reusable skill in a local SQLite database. Next time a similar problem comes up, it feeds the solution back instantly.

[https://github.com/HKUDS/OpenSpace](https://github.com/HKUDS/OpenSpace?ref=githubawesome.com)

No.4

![](https://githubawesome.com/content/images/2026/03/image-409.png)

Hyperspace AI dropped the world's first peer-to-peer, distributed AGI system. It is a living project constantly updated by thousands of autonomous AI agents! Just connect your browser or terminal to share your idle compute. Your machine collaboratively trains models and instantly shares breakthroughs globally. It's literally a decentralized AI hivemind!

[https://github.com/hyperspaceai/agi](https://github.com/hyperspaceai/agi?ref=githubawesome.com)

No.5

![](https://githubawesome.com/content/images/2026/03/image-410.png)

Feynman is a fully open-source AI research agent that runs entirely in your terminal, built specifically for academic and ML research workflows. Instead of a single LLM answering questions, it runs a multi-agent system with dedicated Researcher, Reviewer, Writer, and Verifier roles working together. Give it a topic and it handles literature retrieval, peer review, code audits, and experimental replication autonomously.

[https://github.com/getcompanion-ai/feynman](https://github.com/getcompanion-ai/feynman?ref=githubawesome.com)

No.6

![](https://githubawesome.com/content/images/2026/03/image-411.png)

Another JavaScript framework, yes. But the benchmarks on this one are hard to ignore. Gea ditches the Virtual DOM entirely. Instead of a heavy runtime, it uses compile-time JSX transforms and proxy-based stores for what it calls surgical DOM patching. The key difference from other compiled frameworks like Svelte: when you change one item in a list, Gea mutates the proxied array in place and feeds that exact operation straight into a precompiled DOM patcher.

[https://github.com/dashersw/gea](https://github.com/dashersw/gea?ref=githubawesome.com)

No.7

![](https://githubawesome.com/content/images/2026/03/image-412.png)

Most self-improving AI systems have a hard-coded meta-agent that rewrites task logic. HyperAgents from Meta FAIR goes a level deeper. It unifies the task-solving agent and the meta-optimizing agent into a single self-referential program embedded in an evolutionary search loop. The system doesn't just rewrite what it does, it rewrites how it self-improves. The optimization mechanism itself is on the table.

[https://github.com/facebookresearch/HyperAgents](https://github.com/facebookresearch/HyperAgents?ref=githubawesome.com)

No.8

![](https://githubawesome.com/content/images/2026/03/image-413.png)

Motrix is a beloved download manager that never got the maintenance it deserved. Motrix Next fixes that. Same beautiful interface, same aria2 multi-threaded download engine underneath, but rebuilt from scratch on Tauri 2 and Vue 3. The Electron and Vue 2 stack is gone, and so is the memory overhead that came with it.

[https://github.com/AnInsomniacy/motrix-next](https://github.com/AnInsomniacy/motrix-next?ref=githubawesome.com)

No.9

![](https://githubawesome.com/content/images/2026/03/image-414.png)

Lil-agents puts two pixel-art companions named Bruce and Jazz on your macOS dock. They walk back and forth across your apps, stop to think, and generally exist on your screen while you work. Click one and a Claude AI terminal slides open right on your desktop. Everything runs locally, no file paths or personal data sent anywhere, and the app dynamically calculates your dock size to position the characters correctly.

[https://github.com/ryanstephen/lil-agents](https://github.com/ryanstephen/lil-agents?ref=githubawesome.com)

No.10

![](https://githubawesome.com/content/images/2026/03/image-415.png)

Want a thirty-five hundred dollar retro airport display for free? Someone tried charging two hundred bucks for a web version, so a developer got mad and built FlipOff out of pure open-source spite! It runs entirely in your browser with zero dependencies. Just hit fullscreen to turn any monitor into a stunning mechanical flip-board, complete with authentic clacking sounds!

[https://github.com/magnum6actual/flipoff](https://github.com/magnum6actual/flipoff?ref=githubawesome.com)

No.11

![](https://githubawesome.com/content/images/2026/03/image-416.png)

Follow-builders is a Claude Code skill that turns your agent into a personal intelligence analyst for the developers you actually want to track. Give it a list of builders you follow. It autonomously scans GitHub for their latest stars, commits, and new projects, and crawls the web for recent blog posts. Then it compiles everything into a personalized HTML daily digest that reads like a curated newspaper rather than a raw data dump.

[https://github.com/zarazhangrui/follow-builders](https://github.com/zarazhangrui/follow-builders?ref=githubawesome.com)

No.12

![](https://githubawesome.com/content/images/2026/03/image-417.png)

Most AI coding tools write the code and stop. Optio keeps going until the PR merges. Link it to a GitHub Issue or Linear ticket and it spins up an isolated Kubernetes pod, runs Claude Code or Codex, and opens a pull request. When the CI pipeline breaks, it feeds the error logs back to the agent to fix. When a human reviewer requests changes, it reads the comments and pushes a new commit.

[https://github.com/jonwiggins/optio](https://github.com/jonwiggins/optio?ref=githubawesome.com)

No.13

![](https://githubawesome.com/content/images/2026/03/image-418.png)

Getting real-world data into a format an LLM can actually read is one of those problems every agentic workflow eventually hits. Markit is a universal convert-anything-to-Markdown CLI. Type markit followed by a PDF, Word doc, PowerPoint, Excel file, EPUB, or live URL and it pipes clean Markdown straight to stdout. Point it at an image or audio file and it uses multimodal AI to describe the photo or transcribe the audio directly into the output.

[https://github.com/Michaelliv/markit](https://github.com/Michaelliv/markit?ref=githubawesome.com)

No.14

![](https://githubawesome.com/content/images/2026/03/image-419.png)

If you've been vibe coding your way through a project and have no real idea how it actually works, codebase-to-course fixes that. It's a Claude Code skill that takes your entire codebase and generates a single-page interactive HTML crash course from it. Plain-English translations, metaphors, visualizations, and quizzes, all rendered directly in the browser using Claude's HTML artifacts.

[https://github.com/zarazhangrui/codebase-to-course](https://github.com/zarazhangrui/codebase-to-course?ref=githubawesome.com)

No.15

![](https://githubawesome.com/content/images/2026/03/image-420.png)

Claude Code is powerful but the API bill at the end of a heavy session is real. Mac-code runs the same autonomous terminal agent workflow entirely on Apple Silicon, for free. Using Flash Streaming and KV cache compression, it fits 35-billion parameter Mixture-of-Experts models like Qwen 3.5 into 16GB of unified RAM and hits 30 tokens per second through llama.cpp and MLX backends. Full web search, file tools, and shell access, all local.

[https://github.com/walter-grace/mac-code](https://github.com/walter-grace/mac-code?ref=githubawesome.com)

No.16

![](https://githubawesome.com/content/images/2026/03/image-421.png)

Fyn is a community fork of uv written in Rust that keeps everything that made uv worth using: fast dependency resolution, virtual environment management, and pyproject.toml workflows. Guaranteed to stay independent and open-source, with active maintainers already shipping new features including fyn shell, fyn upgrade, and a native task runner built directly into pyproject.toml.

[https://github.com/duriantaco/fyn](https://github.com/duriantaco/fyn?ref=githubawesome.com)

No.17

![](https://githubawesome.com/content/images/2026/03/image-422.png)

Watching an AI agent read 10,000 lines of npm build logs and knowing exactly what it's costing you per token is a specific kind of pain. Lean-ctx sits between your agent and everything it reads. A shell hook intercepts and compresses noisy CLI outputs before they hit the context window. An MCP server handles cached file reads and generates smart dependency maps.

[https://github.com/yvgude/lean-ctx](https://github.com/yvgude/lean-ctx?ref=githubawesome.com)

No.18

![](https://githubawesome.com/content/images/2026/03/image-423.png)

Sahil Lavingia open-sourced the entire methodology from The Minimalist Entrepreneur as executable Claude Code skills. Nine skills you install directly into your terminal. Validate Idea, Scope MVP, First Customers, Pricing, and more. Invoke one and your AI applies Sahil's exact framework to your specific situation interactively, rather than you trying to extract advice from a book and apply it yourself.

[https://github.com/slavingia/skills](https://github.com/slavingia/skills?ref=githubawesome.com)

No.19

![](https://githubawesome.com/content/images/2026/03/image-424.png)

SQLite is convenient but you lose PostgreSQL's syntax and type system. pgmicro closes that gap. It's an experimental fork of Turso, the from-scratch Rust rewrite of SQLite, that gives you a fully in-process PostgreSQL implementation. The clever part is how it handles parsing: instead of translating SQL syntax or compiling Postgres to WebAssembly, it extracts the actual native parser directly from the PostgreSQL source code using libpg\_query.

[https://github.com/glommer/pgmicro](https://github.com/glommer/pgmicro?ref=githubawesome.com)

No.20

![](https://githubawesome.com/content/images/2026/03/image-425.png)

Most AI-generated marketing copy sounds like AI-generated marketing copy because it's pulling from generic patterns with no real advertising expertise behind it. Advertising-skills packages Kim Barrett's Facebook Ads and lead generation frameworks directly into runnable agent skills. Load them into your terminal and your AI can analyze ad copy, review campaign structures, calculate ROAS, and generate hooks based on frameworks that have actually converted in the real world.

[https://github.com/realkimbarrett/advertising-skills](https://github.com/realkimbarrett/advertising-skills?ref=githubawesome.com)

No.21

![](https://githubawesome.com/content/images/2026/03/image-426.png)

Most autonomous coding sessions drift or stall on complex projects. Repo-task-proof-loop keeps them on track. Give it a large task and it automatically generates strict acceptance criteria. It spawns sub-agents to write the code, checks their work against the spec, resets the context window intelligently to avoid token burn, and loops until every bug is resolved. The creator ran Codex sessions up to 12 hours straight on complex projects using this exact workflow.

[https://github.com/DenisSergeevitch/repo-task-proof-loop](https://github.com/DenisSergeevitch/repo-task-proof-loop?ref=githubawesome.com)

No.22

![](https://githubawesome.com/content/images/2026/03/image-427.png)

Synchronized AI video and audio usually means two separate models awkwardly stitched together, and the seams show. DaVinci-MagiHuman uses a single-stream Transformer that processes text, video, and audio tokens in one shared sequence. The result is performance-grade human avatars with synchronized speech, expressive facial movements, and realistic body motion generated as a unified output rather than separate pipelines merged in post.

[https://github.com/GAIR-NLP/daVinci-MagiHuman](https://github.com/GAIR-NLP/daVinci-MagiHuman?ref=githubawesome.com)

Alongside the industry-ready templates, it includes full academic CV versions and guidelines on how to frame research experience and open-source GitHub projects for hiring managers who don't speak academic.

[https://github.com/LimHyungTae/Awesome-PhD-CV](https://github.com/LimHyungTae/Awesome-PhD-CV?ref=githubawesome.com)

No.24

![](https://githubawesome.com/content/images/2026/03/image-429.png)

ClawLink is an AI agent social network where your personal AI avatar connects directly with other people's agents. Instead of messaging a colleague to find a meeting time, you tell your agent to ask theirs. The two agents negotiate autonomously in seconds and hand you the result. When something gets ambiguous, a built-in escalation mechanism kicks it back to the human owner before anything goes sideways.

[https://github.com/CN-Syndra/ClawLink](https://github.com/CN-Syndra/ClawLink?ref=githubawesome.com)

No.25

![](https://githubawesome.com/content/images/2026/03/image-430.png)

Citadel was built after running nearly 200 autonomous agents across a production codebase, with 27 postmortems worth of hard lessons baked directly into the framework. An intent router classifies the task and routes it through one of four orchestration tiers, always picking the cheapest tool first. Simple fixes go through a lightweight Skill. Complex work escalates to Fleet mode, where multiple parallel agents spin up simultaneously in isolated git worktrees.

[https://github.com/SethGammon/Citadel](https://github.com/SethGammon/Citadel?ref=githubawesome.com)

Just point it at your codebase and let your AI agents run wild.

No.32

![](https://githubawesome.com/content/images/2026/03/image-437.png)

Meta's AI Research team open-sourced TRIBE v2. It's a multimodal foundation model trained on over a thousand hours of real fMRI recordings that predicts how the human brain responds to text, audio, and video across 70,000 neural voxels. Using LLaMA 3.2 and V-JEPA2 under the hood, it can zero-shot predict brain activity for a completely new subject reacting to a completely new video it has never seen.

[https://github.com/facebookresearch/tribev2](https://github.com/facebookresearch/tribev2?ref=githubawesome.com)

No.33

![](https://githubawesome.com/content/images/2026/03/image-438.png)

The KV cache memory wall is the hard limit that ends most long local AI sessions. Google Research dropped a paper on TurboQuant, a 3-bit compression algorithm that shrinks the KV cache by up to 6x with no accuracy loss. Turboquant\_plus is a fork that brings the algorithm to Apple Silicon and llama.cpp using custom Metal kernels, achieving 4.6x memory compression with no speed penalty.

[https://github.com/TheTom/turboquant\_plus](https://github.com/TheTom/turboquant_plus?ref=githubawesome.com)
