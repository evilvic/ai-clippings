It is time for GitHub Trending Weekly number 30. We have tracked down 35 trending repositories on GitHub.

![](https://www.youtube.com/watch?v=obniDiLAuog)

No.1
[3dsvg](https://github.com/renatoworks/3dsvg) — Turns any 2D SVG into an interactive 3D component in your browser. Drag in an SVG, pick from 10 material presets like Glass, Chrome, or Holographic, add animations like floating or spinning, and export as a 4K transparent PNG or 60fps MP4 using FFmpeg WASM running entirely locally. Hit Embed and get a clean React JSX snippet using react-three-fiber ready to drop straight into your Next.js project.
