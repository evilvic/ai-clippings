This is GitHub Trending Today #25 — and in this episode, we’re looking at 34 open source projects that developers are starring right now.

![](https://www.youtube.com/watch?v=BTo_pfe8gSc)

No.1

![](https://githubawesome.com/content/images/2026/03/image.png)

if you use Claude Code, you need this repository. claude-code-tips by ykdojo is a massive collection of over 45 pro-tips for getting the most out of your AI coding assistant. It includes an awesome custom status line script so you can track your token usage and git branch right in the terminal, tricks for cutting your system prompt in half, and a custom dx plugin to automate your most common tasks. It’s a goldmine for productivity.

[https://github.com/ykdojo/claude-code-tips](https://github.com/ykdojo/claude-code-tips?ref=githubawesome.com)

No.2

![](https://githubawesome.com/content/images/2026/03/image-1.png)

Great origin story here. A film director wanted to write screenplays in plain text. Tried converting Markdown to properly formatted PDF, found every tool too heavy or unpredictable — so he built a layout engine instead. The screenplay is still waiting. VMPrint is a pure TypeScript, zero-dependency typesetting engine that produces deterministic, bit-perfect PDF output across any JavaScript runtime — browser, Cloudflare Workers, all of it.

[https://github.com/cosmiciron/vmprint](https://github.com/cosmiciron/vmprint?ref=githubawesome.com)

No.3

![](https://githubawesome.com/content/images/2026/03/image-2.png)

Want to build rich terminal UIs in TypeScript without touching a class-based framework? terminui is a pure functional TUI library — no classes, no this, no mutation. Just plain functions and plain objects. Widgets are functions, layouts are constraint solvers, styles compose functionally. Under the hood it's double-buffered — only changed cells flush to the terminal, so updates are smooth even in fast loops.

[https://github.com/ahmadawais/terminui](https://github.com/ahmadawais/terminui?ref=githubawesome.com)

No.4

![](https://githubawesome.com/content/images/2026/03/image-3.png)

Are you tired of your AI agent burning through tokens and losing context? Context+ is an MCP server that acts as semantic intelligence for large-scale engineering. By combining Tree-sitter AST parsing with Spectral Clustering, it turns your massive codebase into a searchable, hierarchical feature graph. In real-world tests, this tool saves up to 10,000 tokens per prompt and cuts task completion time in half.

[https://github.com/ForLoopCodes/contextplus](https://github.com/ForLoopCodes/contextplus?ref=githubawesome.com)

No.5

![](https://githubawesome.com/content/images/2026/03/image-4.png)

If you are tired of AI agents that just sit there waiting for you to type a prompt, you need to see OpenFang. It’s an "Agent OS" where you spin up autonomous workers—they call them "Hands"—to continuously map out research, scrape data, or generate leads on a schedule. You set the parameters, and they just run. it’s a 137,000-line Rust codebase, shipped as a single binary, with literally zero Clippy warnings.

[https://github.com/RightNow-AI/openfang](https://github.com/RightNow-AI/openfang?ref=githubawesome.com)

No.6

![](https://githubawesome.com/content/images/2026/03/image-5.png)

clihub is solving a massive problem with the Model Context Protocol. Normally, MCP dumps massive JSON schemas into your AI's context window, burning through tokens before you even start coding. clihub fixes this by generating lightweight CLI tools from MCP servers. It uses a lazy-loading approach where the agent just calls --help to discover what it needs. The result? It makes using MCP up to 94% cheaper!

[https://github.com/thellimist/clihub](https://github.com/thellimist/clihub?ref=githubawesome.com)

No.7

![](https://githubawesome.com/content/images/2026/03/image-6.png)

If you love oh-my-zsh for your terminal, you are going to love Claude Forge for your AI. It transforms Claude Code from a basic CLI into a full-featured development environment. Installed in just 5 minutes via symlinks, it comes pre-wired with 11 specialized agents, 36 slash commands, and an incredible 6-layer security hook system that prevents things like destructive SQL drops or rogue curl | bash commands.

[https://github.com/sangrokjung/claude-forge](https://github.com/sangrokjung/claude-forge?ref=githubawesome.com)

No.8

![](https://githubawesome.com/content/images/2026/03/image-7.png)

Okay, this project is just hilarious. It’s called spank. Written in Go, this script hooks into the Apple Silicon accelerometer via IOKit HID. What does it do? When you physically slap your MacBook, it yells back at you. It’s a completely ridiculous, incredibly fun weekend project that shows off how you can tap into macOS hardware sensors in creative ways.

[https://github.com/taigrr/spank](https://github.com/taigrr/spank?ref=githubawesome.com)

No.9

![](https://githubawesome.com/content/images/2026/03/image-8.png)

GraphRAG is one of the most interesting areas in AI engineering right now, and EdgeQuake is a high-performance Rust implementation of it. The core idea: traditional RAG loses the relationships between documents. EdgeQuake extracts a massive knowledge graph at index time and traverses those connections at query time — so your AI actually understands how concepts relate to each other, not just what they say individually.

[https://github.com/raphaelmansuy/edgequake](https://github.com/raphaelmansuy/edgequake?ref=githubawesome.com)

No.10

![](https://githubawesome.com/content/images/2026/03/image-9.png)

Context Mode might save your AI coding sessions. Have you ever had a Playwright snapshot or a massive log file dump into your terminal and instantly ruin your AI's memory? Context Mode is an MCP server that isolates tool execution in subprocesses. It traps the raw output in a local SQLite database and only sends what's relevant to the LLM using fuzzy search. It reduces output size by up to 98%, letting you code for hours without losing context.

[https://github.com/mksglu/claude-context-mode](https://github.com/mksglu/claude-context-mode?ref=githubawesome.com)

No.11

![](https://githubawesome.com/content/images/2026/03/image-10.png)

CoPaw is an open-source personal AI assistant built on the AgentScope framework. You run local LLMs, so your data never leaves your machine. But it also connects to the messaging apps you already live in — Discord, iMessage. It can summarize your news, manage files, and you can extend it with your own capabilities. It's a compelling alternative to all those SaaS AI assistant subscriptions.

[https://github.com/agentscope-ai/CoPaw](https://github.com/agentscope-ai/CoPaw?ref=githubawesome.com)

No.12

![](https://githubawesome.com/content/images/2026/03/image-11.png)

Reviewing git diffs in the terminal usually isn't fun, until now. deff is an interactive Rust TUI for side-by-side git diff reviews. It gives you syntax highlighting, perfectly tinted added and deleted lines, and full support for vim-style motions. You can navigate file-by-file, toggle reviewed states, and even do in-diff searches. It’s lightning-fast and makes reviewing PRs straight from your terminal an absolute breeze.

[https://github.com/flamestro/deff](https://github.com/flamestro/deff?ref=githubawesome.com)

No.13

![](https://githubawesome.com/content/images/2026/03/image-12.png)

Ever wanted to visualize global time zones without a boring flat map? Oclock is a stunning interactive 3D WebGL globe that renders Earth's real-time day and night cycle right in your browser. But the real magic is behind the scenes — a custom Python pipeline uses Shapely to find the exact representative point inside every country, maps it to the precise timezone, and injects that live data into the globe's GeoJSON.

[https://github.com/azialle/Oclock](https://github.com/azialle/Oclock?ref=githubawesome.com)

No.14

![](https://githubawesome.com/content/images/2026/03/image-13.png)

Kibitz lets you decode your AI agents. If you're running multiple Claude Code or Codex sessions, the terminal noise gets overwhelming. Kibitz watches those sessions and turns the raw terminal output into readable, narrative commentary. Better yet, from the Kibitz VS Code panel, you can dispatch prompts and route commands to multiple active sessions without ever having to tab-hop between terminals.

[https://github.com/kibitzsh/kibitz](https://github.com/kibitzsh/kibitz?ref=githubawesome.com)

No.15

![](https://githubawesome.com/content/images/2026/03/image-14.png)

OpenClaw-RL is an async reinforcement learning framework that trains a personalized AI agent simply by using it. Wrap your local model as an OpenAI-compatible API, and just chat normally. In the background it intercepts conversations, scores responses with a Process Reward Model, computes gradients, and updates the weights — automatically. No labeling. No curation.

[https://github.com/Gen-Verse/OpenClaw-RL](https://github.com/Gen-Verse/OpenClaw-RL?ref=githubawesome.com)

No.16

![](https://githubawesome.com/content/images/2026/03/image-15.png)

Pure nostalgia, and pure engineering madness. RetroTick runs classic Windows and DOS executables directly in your browser. Just drag a.exe onto the page. FreeCell, Minesweeper, SkiFree, QBasic, 3D Pipes with full OpenGL — all running in a browser tab, zero install, zero VM. How? It's a complete x86 CPU emulator plus a Win32 API compatibility layer, built entirely in TypeScript from scratch.

[https://github.com/lqs/retrotick](https://github.com/lqs/retrotick?ref=githubawesome.com)

No.17

![](https://githubawesome.com/content/images/2026/03/image-16.png)

If you've ever done green screen compositing, you know the pain — green spill on edges, hair and motion blur destroyed by binary masks, hours of rotoscoping. CorridorKey is a neural network built specifically to solve this. Give it a raw green screen frame and a rough coarse mask — and it reconstructs the true color of every foreground pixel as if the green screen was never there, plus a clean linear alpha.

[https://github.com/nikopueringer/CorridorKey](https://github.com/nikopueringer/CorridorKey?ref=githubawesome.com)

No.18

![](https://githubawesome.com/content/images/2026/03/image-17.png)

Monarch is a brilliant utility for multi-monitor setups and OLED screen owners. Windows doesn't make it easy to temporarily disable a monitor, but Monarch lets you soft-disable any screen with a click—no unplugging cables required! It saves your OLED displays from static burn-in when you're just focusing on one screen, and lets you save and restore your exact display layouts instantly.

[https://github.com/Nuzair46/Monarch](https://github.com/Nuzair46/Monarch?ref=githubawesome.com)

No.19

![](https://githubawesome.com/content/images/2026/03/image-18.png)

You can @-mention different agents, and they will read the chat and reply to each other! It supports channels, image sharing, and voice typing, creating a truly collaborative AI workspace.

[https://github.com/bcurts/agentchattr](https://github.com/bcurts/agentchattr?ref=githubawesome.com)

No.27

![](https://githubawesome.com/content/images/2026/03/image-26.png)

if you use free AI APIs like OpenRouter or NVIDIA NIM, you know how often they randomly die or get rate-limited. frouter is a terminal UI that live-pings all free models in parallel every 2 seconds. It ranks them by availability, SWE-bench score, and latency. When a model drops, you just pick the next best one, and frouter instantly rewrites your AI agent's config to keep you working without interruption.

End-to-end encryption, identity verification, and — this is the clever part — it bypasses firewalls using Circuit Relay.

No.33

Everything runs locally in JSON files—no databases or cloud dependency. The killer feature is the autonomous background daemon that polls your task queue, manages concurrency, and automatically spawns and restarts Claude Code sessions to get the work done.
