It is time for GitHub Trending Today, episode number 26. Today we have 34 trending open-source projects on GitHub.

![](https://www.youtube.com/watch?v=Fq7YnzT13sA)

No.1

![](https://githubawesome.com/content/images/2026/03/image-100.png)

Ever wanted to make a custom map poster of your city? Terraink is a map poster engine that runs entirely in your browser. built with React that uses MapLibre and OpenStreetMap data. You pick a location, customize colors, roads, water, parks, and typography — and export a print-ready PNG. It's inspired by MapToPoster but rebuilt from scratch with a full theme system and Docker support for self-hosting.

[https://github.com/yousifamanuel/terraink](https://github.com/yousifamanuel/terraink?ref=githubawesome.com)

No.2

![](https://githubawesome.com/content/images/2026/03/image-101.png)

If your AI coding agent keeps forgetting everything after its context gets compacted, Engram is the fix. It gives your AI persistent memory using a single Go binary with SQLite and FTS5. The agent proactively saves its state after major architecture decisions or bug fixes, and can instantly recover its full session state when needed. No more repeating yourself!

[https://github.com/Gentleman-Programming/engram](https://github.com/Gentleman-Programming/engram?ref=githubawesome.com)

No.3

![](https://githubawesome.com/content/images/2026/03/image-102.png)

Want to run an entire team of AI agents from a single coding session? Overstory does exactly that. It spawns worker agents in isolated Git worktrees using tmux, coordinates them through a lightning-fast SQLite mail system, and safely merges their work back with tiered conflict resolution. Supports Claude Code, Pi, and Gemini CLI out of the box. One session, a whole swarm.

[https://github.com/jayminwest/overstory](https://github.com/jayminwest/overstory?ref=githubawesome.com)

No.4

![](https://githubawesome.com/content/images/2026/03/image-103.png)

From the Open WebUI team — Open Terminal. AI assistants are great at writing code, but they need somewhere to actually run it. Open Terminal gives your AI a real, sandboxed operating system via Docker. Instead of just writing text, your LLM gets full shell access to install software, run Python scripts, and manage files—all completely isolated from your host machine!

[https://github.com/open-webui/open-terminal](https://github.com/open-webui/open-terminal?ref=githubawesome.com)

No.5

![](https://githubawesome.com/content/images/2026/03/image-104.png)

Stop dumping massive 800-line files into your AI's context window! jCodeMunch is a brilliant new MCP server that indexes your entire codebase once using tree-sitter AST parsing. Instead of forcing Claude or VS Code to read entire files, the AI can ask to retrieve exact functions or classes by their symbol! It drops context usage from 40,000 tokens down to just 200, making AI cheap and hyper-efficient.

[https://github.com/jgravelle/jcodemunch-mcp](https://github.com/jgravelle/jcodemunch-mcp?ref=githubawesome.com)

No.6

![](https://githubawesome.com/content/images/2026/03/image-105.png)

Are you ready to hire a tireless AI researcher? Coming from the legendary Andrej Karpathy, autoresearch is a fully automated AI lab in a few Python scripts. You point it to a model and go to bed. The AI autonomously modifies the training architecture, runs a 5-minute training loop on your GPU, checks the validation loss, and decides whether to keep or discard the changes! By morning, you wake up to a newly optimized model.

[https://github.com/karpathy/autoresearch](https://github.com/karpathy/autoresearch?ref=githubawesome.com)

No.7

![](https://githubawesome.com/content/images/2026/03/image-106.png)

If you use AI to build frontend apps, every interface comes out looking like the exact same generic template. Uncodixfy fixes this with a brilliantly simple hack. It is a single markdown file packed with strict rules that forbid your AI from defaulting to cliché UI patterns! Just drop it into your prompt, and it blocks the AI's bad habits, pushing it to generate genuinely professional, unique designs.

[https://github.com/cyxzdev/Uncodixfy](https://github.com/cyxzdev/Uncodixfy?ref=githubawesome.com)

No.8

![](https://githubawesome.com/content/images/2026/03/image-107.png)

Need to search your Discord server history without relying on Discord's broken built-in search? discrawl is a Go CLI that mirrors an entire Discord guild — channels, threads, and full message history — into a local SQLite database with full-text search. Tail live Gateway events for real-time sync, run raw SQL queries against your archive, or enable OpenAI embeddings for semantic search.

[https://github.com/steipete/discrawl](https://github.com/steipete/discrawl?ref=githubawesome.com)

No.9

![](https://githubawesome.com/content/images/2026/03/image-108.png)

Antigravity Claw makes any webpage defy gravity. Point it at a URL, and this Rust physics engine wrapped in a Python CLI fetches the DOM, extracts every element, runs a real Verlet integration simulation with collision detection, and outputs an interactive HTML animation you can open in any browser. Classic Google Gravity vibes — or enable chaos mode for maximum entropy.

[https://github.com/deborahikssv/Antigravity-claw](https://github.com/deborahikssv/Antigravity-claw?ref=githubawesome.com)

No.10

![](https://githubawesome.com/content/images/2026/03/image-109.png)

Think Lovable or Bolt.new, but for mobile — and fully open source. Vibra Code lets you describe a React Native app in English, then Claude Code running inside an E2B cloud sandbox generates it while you watch the preview update live on your actual phone. 60fps iOS chat UI, backend on Next.js, Convex, and Inngest. The complete source code is right there to fork and self-host.

[https://github.com/sa4hnd/vibra-code](https://github.com/sa4hnd/vibra-code?ref=githubawesome.com)

No.11

![](https://githubawesome.com/content/images/2026/03/image-110.png)

Kula is a self-contained Linux server monitoring tool that reads directly from proc and sys every second. Single Go binary, zero external dependencies, no databases. Tracks per-core CPU, memory, swap, network throughput, disk I/O, process states, and clock sync, storing everything in a tiered ring-buffer with bounded disk usage. Serves a real-time web dashboard and terminal TUI.

[https://github.com/c0m4r/kula](https://github.com/c0m4r/kula?ref=githubawesome.com)

No.12

![](https://githubawesome.com/content/images/2026/03/image-111.png)

Get ready for the most cursed and brilliant project of the week: A Rust compiler written entirely in PHP! rustc-php takes raw Rust source code and directly emits x86-64 Linux binaries with no LLVM, no assembler, and no linker required. In just 2,500 lines of PHP, it actually implements ownership checking, borrow checking, and mutable references. Absolutely unhinged project.

[https://github.com/mrconter1/rustc-php](https://github.com/mrconter1/rustc-php?ref=githubawesome.com)

No.13

![](https://githubawesome.com/content/images/2026/03/image-112.png)

You can step through the AI's execution, expand its tool calls, and see exactly what files it touched. It generates a single HTML file, making it the ultimate way to share complex AI debugging sessions with your team.

[https://github.com/es617/claude-replay](https://github.com/es617/claude-replay?tab=readme-ov-file&ref=githubawesome.com)

No.14

![](https://githubawesome.com/content/images/2026/03/image-113.png)

Symphony-ts is the community's answer: a complete TypeScript port bringing the exact same autonomous worker orchestration, CI checking, and PR generation into the familiar Node.js ecosystem. All the power, now in a stack you can actually hack on yourself.

[https://github.com/OasAIStudio/symphony-ts](https://github.com/OasAIStudio/symphony-ts?ref=githubawesome.com)

No.15

![](https://githubawesome.com/content/images/2026/03/image-114.png)

When browsers killed plugins, thousands of early 2000s web games were lost forever. DirPlayer brings them back. This team reverse-engineered Macromedia Shockwave and rebuilt the entire engine from scratch in Rust and WebAssembly. No GC pauses, full memory safety, hardware-accelerated graphics. Classics like Habbo Hotel and LEGO Junkbot — playable natively in your browser again.

[https://github.com/igorlira/dirplayer-rs/](https://github.com/igorlira/dirplayer-rs/?ref=githubawesome.com)

No.16

![](https://githubawesome.com/content/images/2026/03/image-115.png)

Your Twitter bookmarks are a black hole — thousands of saved tweets you never look at again. Siftly fixes that. It's a local-first bookmark organizer built in TypeScript that uses AI to automatically categorize and tag every saved tweet, then generates gorgeous interactive mindmap visualizations so you can actually explore your saved knowledge visually. Your data stays local, your chaos gets structured.

[https://github.com/viperrcrypto/Siftly](https://github.com/viperrcrypto/Siftly?ref=githubawesome.com)

No.17

![](https://githubawesome.com/content/images/2026/03/image-116.png)

WorldFM is a real-time multi-view diffusion model. Unlike AI video generators that just morph pixels, this is an open-source generative frame model designed specifically for spatial intelligence. Give it a single photo or text prompt, and it generates a consistent, interactive 3D world you can actually walk around in! Best of all, it is highly optimized to run in real-time on consumer hardware like an RTX 4090.

[https://github.com/inspatio/worldfm](https://github.com/inspatio/worldfm?ref=githubawesome.com)

![](https://githubawesome.com/content/images/2026/03/image-117.png)

No.19

No.20

![](https://githubawesome.com/content/images/2026/03/image-119.png)

Pure nostalgia. AnsiSaver is a macOS screensaver that pulls ANSI art directly from 16colo.rs — the largest BBS art archive on the internet — and scrolls it across your screen at 60fps. Authentic CP437 fonts, neon color palettes, rendered with the same libansilove library the archive uses. Like leaving your terminal connected to a BBS you never logged off from. Scroll speed, scale, pack URLs — all configurable.

[https://github.com/lardissone/ansi-saver](https://github.com/lardissone/ansi-saver?ref=githubawesome.com)

No.21

![](https://githubawesome.com/content/images/2026/03/image-120.png)

Unredact is a computer vision tool that literally unmasks redacted names in classified documents. It analyzes the exact geometry of black redaction boxes — measuring font spacing, character width, and surrounding typography — then cross-references known datasets to calculate exactly which names physically fit the space. The author is very clear: this is for research and entertainment only.

[https://github.com/Alex-Gilbert/unredact](https://github.com/Alex-Gilbert/unredact?ref=githubawesome.com)

Instead of parsing raw text, it intelligently highlights the exact visual, structural, and textual differences between two files. Legal contracts, academic papers, design specs — hours of manual proofreading, gone.

No.24
