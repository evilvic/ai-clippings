This is GitHub Trending Today #24 — 31 projects, just what developers are building and starring right now.

![](https://www.youtube.com/watch?v=ppC9a5-Cle0)

No. 1 — pinchtab [https://github.com/pinchtab/pinchtab](https://github.com/pinchtab/pinchtab?ref=githubawesome.com)

![](https://githubawesome.com/content/images/2026/02/image-222.png)

Most browser automation tools are locked inside their own frameworks — if you switch agents, you start over. Pinchtab is a standalone HTTP server that gives any AI agent, language, or tool control over a Chrome browser via API.Accessibility-first with structured element trees, stealth mode for bot detection bypass, and smart filtering that reduces token usage to as low as 800 tokens per page.

No. 2 — OpenPlanter [https://github.com/ShinMegamiBoson/OpenPlanter](https://github.com/ShinMegamiBoson/OpenPlanter?ref=githubawesome.com)

![](https://githubawesome.com/content/images/2026/02/image-223.png)

OpenPlanter is a recursive investigation agent with a terminal UI that ingests corporate registries, campaign finance records, lobbying disclosures, and government contracts. Resolves entities across sources and surfaces connections through evidence-backed analysis. Runs sub-agents in parallel for large investigations. Full file I/O, shell execution, web search, and recursive delegation.

No. 3 — apple-silicon-accelerometer [https://github.com/olvvier/apple-silicon-accelerometer](https://github.com/olvvier/apple-silicon-accelerometer?ref=githubawesome.com)

![](https://githubawesome.com/content/images/2026/02/image-224.png)

Your Apple Silicon MacBook has a MEMS accelerometer buried inside—Apple never documented a public API. This project reads raw three-axis acceleration data at around 800Hz by going directly through IOKit HID callbacks to the undocumented AppleSPU sensor. Practical use is vibration detection, but the demo is what gets you: place your wrists near the trackpad and it uses ballistocardiography to estimate your heartbeat from chassis vibrations.

No. 4 — visual-json [https://github.com/vercel-labs/visual-json](https://github.com/vercel-labs/visual-json?ref=githubawesome.com)

![](https://githubawesome.com/content/images/2026/02/image-225.png)

JSON editors haven't changed much in years. This is Vercel Labs' attempt to change that. Visual JSON is a schema-aware, embeddable, extensible JSON editor with a headless core and React UI components — TreeView, FormView, DiffView, and more. Drop it into your admin panel, your CMS, your developer tool. It's schema-driven so it knows what valid values look like, and it's built to be extended.

No. 6 — MapToPoster JS [https://github.com/dimartarmizi/map-to-poster](https://github.com/dimartarmizi/map-to-poster?ref=githubawesome.com)

![](https://githubawesome.com/content/images/2026/02/image-227.png)

No. 15 — GNL — Garment Notation Language [https://github.com/khalildh/garment-notation](https://github.com/khalildh/garment-notation?ref=githubawesome.com)

![](https://githubawesome.com/content/images/2026/02/image-236.png)

No. 17 — btrpa-scan [https://github.com/HackingDave/btrpa-scan](https://github.com/HackingDave/btrpa-scan?ref=githubawesome.com)

![](https://githubawesome.com/content/images/2026/02/image-238.png)

From David Kennedy at TrustedSec — a Python BLE scanner with Resolvable Private Address resolution. Bluetooth devices use randomized addresses to prevent tracking, but if you have the device's Identity Resolving Key, those addresses can be resolved back to a real identity. btrpa-scan implements the full ah function from the Bluetooth Core Specification to do this in real time.

No. 18 — agent-vault [https://github.com/botiverse/agent-vault](https://github.com/botiverse/agent-vault?ref=githubawesome.com)

![](https://githubawesome.com/content/images/2026/02/image-239.png)

When you let an AI agent set up a service that needs an API key, that key flows through the LLM provider's servers. Agent-vault prevents that. It acts as a secret-aware file I/O layer — agents see placeholders like agent-vault:openai-key instead of real values. When the agent writes a config file with that placeholder, agent-vault substitutes the real secret before it hits disk. When the agent reads the file back, the real value is redacted again.

No. 23 — shuru [https://github.com/superhq-ai/shuru](https://github.com/superhq-ai/shuru?ref=githubawesome.com)

![](https://githubawesome.com/content/images/2026/02/image-244.png)

AI agents running on your local machine are powerful. They're also terrifying if they can access your entire filesystem. Shuru solves that. It's a local-first microVM sandbox for macOS — boots lightweight Linux VMs using Apple's Virtualization framework. Each sandbox is ephemeral: the root filesystem resets on every run, giving agents a disposable environment to execute code, and run tools without ever touching your host machine.

No. 24 — walkie [https://github.com/vikasprogrammer/walkie](https://github.com/vikasprogrammer/walkie?ref=githubawesome.com)

![](https://githubawesome.com/content/images/2026/02/image-245.png)

AI agents are isolated by default. When two agents on different machines need to collaborate — or even just coordinate — there's no simple primitive for that. Walkie gives them a walkie-talkie. Pick a channel name, share a secret, and agents find each other automatically over the internet via Hyperswarm DHT — fully peer-to-peer, no relay server, no setup. All communication is encrypted via the Noise protocol.
