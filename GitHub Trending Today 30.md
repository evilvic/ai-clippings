This is GitHub Trending Today, Episode 30, and we are not slowing down. 35 trending open-source projects on GitHub.

![](https://www.youtube.com/watch?v=PEhOF4vGZas)

No.1
[clicky](https://github.com/farzaa/clicky) — AI buddy that floats next to your cursor, watches your screen, speaks to you, and points at elements directly on your monitor. Built by Farza (buildspace founder), fully open-sourced.

No.2
[Locker](https://github.com/zmeyer44/Locker) — Open-source file storage and knowledge platform. Self-hostable alternative to Dropbox/Google Drive with built-in search, document transcription, and AI-powered knowledge base.

No.3
[graphify](https://github.com/safishamsi/graphify) — AI coding assistant skill. Turn any folder of code, docs, papers, images, or videos into a queryable knowledge graph.

No.4
[qmd](https://github.com/tobi/qmd) — Mini CLI search engine for your docs, knowledge bases, meeting notes, whatever.

No.5
[parlor](https://github.com/fikrikarim/parlor) — On-device, real-time multimodal AI. Natural voice and vision conversations running entirely on your machine.

No.6
[mini-coding-agent](https://github.com/rasbt/mini-coding-agent) — Minimal and readable coding agent harness implementation in Python to explain the core components of coding agents.

No.7
[gemma-gem](https://github.com/kessler/gemma-gem) — Runs Google's Gemma 4 model entirely on-device via WebGPU. No API keys, no cloud, no data leaving your machine.

No.8
[clawchief](https://github.com/snarktank/clawchief) — Turn your OpenClaw into a Chief of Staff.

No.9
[KarpathyTalk](https://github.com/karpathy/KarpathyTalk) — Positive developer community for builders and agents. Twitter meets GitHub Gists, with posts in plain Markdown and open data accessible by humans and LLMs via API.

No.10
[badclaude](https://github.com/GitFrog1111/badclaude) — Sometimes Claude Code is going too slow, and you must whip him into shape.

No.11
[rtk](https://github.com/rtk-ai/rtk) — CLI proxy that reduces LLM token consumption by 60-90% on common dev commands. Single Rust binary, zero dependencies.

No.12
[CodeIsland](https://github.com/wxtsky/CodeIsland) — Real-time AI coding agent status panel for macOS Dynamic Island (Notch).

No.13
[Falcon-Perception](https://github.com/tiiuae/Falcon-Perception) — Inference repo for Falcon-Perception and Falcon-OCR model. Early-fusion, natively multimodal, dense Autoregressive Transformer models.

No.14
[clauditor](https://github.com/IyadhKhalfallah/clauditor) — Stop Claude Code from burning through your quota in 20 minutes. Auto-rotates oversized sessions and preserves context.

No.15
[buddy-reroll](https://github.com/grayashh/buddy-reroll) — Reroll your Claude Code /buddy companion to any species, rarity, eye, hat, and shiny combo.

No.16
[OpenMontage](https://github.com/calesthio/OpenMontage) — The first open-source, agentic video production system.

No.17
[gnhf](https://github.com/kunchenguid/gnhf) — "Good night, have fun." Orchestrator that enables autonomous agents to work on code improvements while you sleep.

No.18
[mathcode](https://github.com/math-ai-org/mathcode) — MathCode: A Frontier Mathematical Coding Agent.

No.19
[ghost-pepper](https://github.com/matthartman/ghost-pepper) — Hold-to-talk speech-to-text for macOS. 100% local, powered by WhisperKit and local LLM cleanup. Hold Control to record, release to transcribe and paste.

No.20
[GuppyLM](https://github.com/arman-bd/guppylm) — A ~9M parameter LLM that talks like a small fish.

No.21
[tailslayer](https://github.com/LaurieWired/tailslayer) — Library for reducing tail latency in RAM reads.

No.22
[macpow](https://github.com/k06a/macpow) — Real-time power tree TUI for Apple Silicon.

No.23
[sage-wiki](https://github.com/xoai/sage-wiki) — LLM-compiled personal knowledge base. Drop in papers, articles, notes — compiles into structured, interlinked wiki with concepts extracted and cross-references discovered.

No.24
[fieldtheory-cli](https://github.com/afar1/fieldtheory-cli) — Sync and locally store all your X/Twitter bookmarks. Free and open source CLI for Mac.

No.25
[three-man-team](https://github.com/russelleNVy/three-man-team) — Structured 3-agent AI dev team: Architect, Builder, Reviewer. Token-optimized. Works with Claude Code, VS Code, Cursor.

No.26
[gemma-tuner-multimodal](https://github.com/mattmireles/gemma-tuner-multimodal) — Fine-tune Gemma 4 and 3n with audio, images and text on Apple Silicon using PyTorch and Metal Performance Shaders.

No.27
[storm](https://github.com/orchetron/storm) — Compositor-based terminal UI framework. Fast. Layered. Unstoppable.

No.28
[revdiff](https://github.com/umputun/revdiff) — TUI for reviewing diffs, files, and documents with inline annotations.

No.29
[textual-diff-view](https://github.com/batrachianai/textual-diff-view) — Textual widget for beautiful diffs in the terminal.

No.30
[handoff](https://github.com/pion/handoff) — Start WebRTC in the browser, run it somewhere else.

No.31
[envradar](https://github.com/CodMughees/envradar) — Catch undocumented, unused, and drifting environment variables in your repo.

No.32
[loophole](https://github.com/brendanhogan/loophole) — Adversarial moral-legal code system. AI tool that stress-tests ethical principles, system prompts, and legal texts by trying to break them.

No.33
[slopc](https://github.com/shorwood/slopc) — Proc macro that uses a hallucination machine to write your function bodies at compile time.

No.34
[claude-skill-app-onboarding-questionnaire](https://github.com/adamlyttleapps/claude-skill-app-onboarding-questionnaire) — Claude Code skill that designs and builds high-converting questionnaire-style onboarding flows for your app.

No.35
[skwd-wall](https://github.com/liixini/skwd-wall) — Aesthetics-first wallpaper selector supporting images, videos and Wallpaper Engine Scenes with matugen theme generation.
