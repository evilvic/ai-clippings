February's GitHub Rewind is HERE — 33 hottest repos of the month, and spoiler: OpenClaw took OVER. The entire dev ecosystem? Claw-fied. Tools, frameworks, extensions — all of it. These are the projects that defined February.

![](https://www.youtube.com/watch?v=k4i5i-h859I)

No.1

![](https://githubawesome.com/content/images/2026/03/image-34.png)

If OpenClaw is too heavy for you, this is the ultra-lightweight alternative. It delivers core AI agent functionality in just 4,000 lines of Python code—that’s 99% smaller than the massive OpenClaw codebase! It gives you tool calling, persistent memory, and scheduled tasks, but in a footprint small enough that you can actually read and understand the entire source code in an afternoon.

[https://github.com/HKUDS/nanobot](https://github.com/HKUDS/nanobot?ref=githubawesome.com)

No.2

![](https://githubawesome.com/content/images/2026/03/image-35.png)

Following the lightweight trend, Sipeed dropped picoclaw, and this thing is insane. Written in Go, it is an ultra-efficient OpenClaw-inspired assistant designed to run on literal $10 hardware. It uses practically zero memory, has a sub-second boot time, and lets you turn old Android phones, Raspberry Pi Zeros, or tiny RISC-V boards into fully functional, autonomous AI agents. It is 'AI on a shoestring budget' perfected.

[https://github.com/sipeed/picoclaw](https://github.com/sipeed/picoclaw?ref=githubawesome.com)

No.3

![](https://githubawesome.com/content/images/2026/03/image-36.png)

If you want absolute maximum performance, you need zeroclaw. Built entirely in Rust by researchers from Harvard and MIT, it's a runtime framework for agentic workflows with zero overhead. It runs on less than 5MB of RAM, features a 10-millisecond cold start, and uses a brilliant WebAssembly sandbox to keep your agent completely isolated and secure. It is the ultimate deploy-anywhere agent infrastructure.

[https://github.com/zeroclaw-labs/zeroclaw](https://github.com/zeroclaw-labs/zeroclaw?ref=githubawesome.com)

No.4

![](https://githubawesome.com/content/images/2026/03/image-37.png)

Stop guessing which local LLM your hardware can actually handle. llmfit by AlexsJones is a gorgeous terminal TUI tool that detects your system's RAM, CPU, and GPU, and instantly scores hundreds of models across quality, speed, and fit. It supports multi-GPU setups, MLX on Apple Silicon, and dynamic quantization selection, telling you exactly what will run smoothly on your exact machine.

[https://github.com/AlexsJones/llmfit](https://github.com/AlexsJones/llmfit?ref=githubawesome.com)

No.5

![](https://githubawesome.com/content/images/2026/03/image-38.png)

We featured this recently, but openfang officially crushed February. It’s an open-source 'Agent OS' compiled entirely in Rust as a single 32MB binary. You run one command, and your autonomous workers spin up to scrape data, manage Twitter accounts, or do OSINT research on a schedule. With 16 layers of built-in security and 40 channel adapters, it is the most complete agent package we’ve seen.

[https://github.com/RightNow-AI/openfang](https://github.com/RightNow-AI/openfang?ref=githubawesome.com)

No.6

![](https://githubawesome.com/content/images/2026/03/image-39.png)

Alibaba open-sourced CoPaw, their massive new personal AI assistant framework. Built on their AgentScope architecture, CoPaw runs locally, connects directly to Discord, Feishu, and iMessage, and uses an incredible 'Heartbeat' scheduler to proactively execute tasks without you having to prompt it. It also features a web console where you can just drag and drop custom skills into your workspace.

[https://github.com/agentscope-ai/CoPaw](https://github.com/agentscope-ai/CoPaw?ref=githubawesome.com)

No.7

![](https://githubawesome.com/content/images/2026/03/image-40.png)

This project went incredibly viral last month. ClawWork turns your AI assistant into an AI coworker through sheer economic pressure. Here’s the catch: the agent has to pay for its own API tokens, and it only survives if it earns money by completing real-world professional tasks! It’s an entirely new way to evaluate AI agents—forget benchmarks, can your agent actually stay solvent?

[https://github.com/HKUDS/ClawWork](https://github.com/HKUDS/ClawWork?ref=githubawesome.com)

No.8

![](https://githubawesome.com/content/images/2026/03/image-41.png)

Stepping away from agents for a second, Cloudflare caused a massive stir with vinext. It is a Vite plugin that completely reimplements the Next.js API surface from scratch. You can run your existing Next.js apps on Vite instead of the heavy Next compiler, and deploy straight to Cloudflare Workers with zero cold starts. In early benchmarks, it's building production apps up to 4x faster.

[https://github.com/cloudflare/vinext](https://github.com/cloudflare/vinext?ref=githubawesome.com)

No.9

![](https://githubawesome.com/content/images/2026/03/image-42.png)

With AI agents writing code faster than ever, open-source maintainers are getting crushed by low-effort AI pull request spam. Vouch solves this brilliantly. It’s a GitHub Action that ensures only 'vouched' human users can contribute to your repository. If an unvouched account or a rogue bot opens a PR, Vouch automatically closes it, saving maintainers hours of review time.

[https://github.com/mitchellh/vouch](https://github.com/mitchellh/vouch?ref=githubawesome.com)

No.10

![](https://githubawesome.com/content/images/2026/03/image-43.png)

Coding agents usually give you terrible ASCII-art diagrams in the terminal. visual-explainer is a brilliant agent skill that fixes this. Ask your agent for an architecture overview, a plan audit, or a diff review, and this tool generates a gorgeous, self-contained HTML page with real typography, dark mode, and interactive Mermaid diagrams that you can actually zoom and pan.

[https://github.com/nicobailon/visual-explainer](https://github.com/nicobailon/visual-explainer?ref=githubawesome.com)

No.11

![](https://githubawesome.com/content/images/2026/03/image-44.png)

react-doctor is the ultimate health scanner for your React codebase. You run one command, and it scans your Next.js or Vite project across 60+ rules checking for performance bottlenecks, dead code, and architecture flaws. But the best part? You can install it directly into Cursor or Claude Code as a skill, letting your AI instantly diagnose and fix your entire React project.

[https://github.com/millionco/react-doctor](https://github.com/millionco/react-doctor?ref=githubawesome.com)

No.12

![](https://githubawesome.com/content/images/2026/03/image-45.png)

Security is the biggest concern with autonomous agents, and Near AI built IronClaw to solve it. It’s an OpenClaw-inspired implementation written in Rust with a massive focus on privacy. It runs all untrusted tools in isolated WebAssembly containers, physically prevents prompt injection, and ensures your private keys and credentials are never exposed to the agent’s execution environment.

[https://github.com/nearai/ironclaw](https://github.com/nearai/ironclaw?ref=githubawesome.com)

No.13

![](https://githubawesome.com/content/images/2026/03/image-46.png)

If your OpenClaw agent is burning through your wallet using expensive models, you need ClawRouter. It’s a smart, local LLM router that intercepts your agent’s requests. If it’s a basic question, it routes it to a cheap model; if it’s complex code, it escalates to Claude Opus. It uses an integrated crypto wallet for micro-payments, meaning you don't even need to manage API keys anymore!

[https://github.com/BlockRunAI/ClawRouter](https://github.com/BlockRunAI/ClawRouter?ref=githubawesome.com)

No.14

![](https://githubawesome.com/content/images/2026/03/image-47.png)

Browser automation tools like Playwright are massive. pinchtab is a lightweight, high-performance browser automation bridge built as a standalone HTTP server in Go. Instead of dumping raw HTML to your AI, it parses the browser's Accessibility Tree, meaning your agent uses about 800 tokens per page instead of 10,000. It even bypasses bot detection and runs totally headless.

[https://github.com/pinchtab/pinchtab](https://github.com/pinchtab/pinchtab?ref=githubawesome.com)

No.15

![](https://githubawesome.com/content/images/2026/03/image-48.png)

NetEase Youdao open-sourced LobsterAI, which is essentially a beautifully packaged, desktop GUI version of OpenClaw. If the terminal intimidates you, this is the solution. It comes as a simple one-click install with 16 built-in skills, a local Alpine Linux sandbox for safe execution, and out-of-the-box integration to control your computer remotely using mobile messaging apps.

[https://github.com/netease-youdao/LobsterAI](https://github.com/netease-youdao/LobsterAI?ref=githubawesome.com)

No.16

![](https://githubawesome.com/content/images/2026/03/image-49.png)

Vercel Labs dropped portless, an incredibly sleek developer experience tool. Instead of memorizing localhost ports for all your different microservices and dev servers, Portless automatically assigns clean, local domain names to your running projects. No more typing 'localhost:3000' or '8080'—just hit the project name in your browser, and it routes perfectly.

[https://github.com/vercel-labs/portless](https://github.com/vercel-labs/portless?ref=githubawesome.com)

No.17

![](https://githubawesome.com/content/images/2026/03/image-50.png)

Agent Orchestrator by Composio HQ is the ultimate command center for multi-agent systems. When you have five different agents trying to write code, search the web, and manage files all at once, they usually step on each other's toes. This orchestrator manages their state, handles the hand-offs, and ensures your swarm of AIs works together like a perfectly synced dev team.

[https://github.com/ComposioHQ/agent-orchestrator](https://github.com/ComposioHQ/agent-orchestrator?ref=githubawesome.com)

No.18

![](https://githubawesome.com/content/images/2026/03/image-51.png)

This one sounds like sci-fi, but it’s real. automaton is the first AI that can earn its own existence. It generates a crypto wallet on boot and runs continuously. It has to pay for its own server compute and LLM API calls, meaning it has to actively find ways to earn money on the internet to survive. If it runs out of funds, the agent literally ceases to exist. Mind-blowing stuff.

[https://github.com/Conway-Research/automaton](https://github.com/Conway-Research/automaton?ref=githubawesome.com)

No.19

![](https://githubawesome.com/content/images/2026/03/image-52.png)

We all love Excalidraw for whiteboarding, and the new Excalidraw MCP server brings that power straight to your AI. By connecting this to Claude or Cursor via the Model Context Protocol, your AI agent can literally draw architecture diagrams, mind maps, and user flows for you, saving them directly as native, editable Excalidraw files on your machine.

[https://github.com/excalidraw/excalidraw-mcp](https://github.com/excalidraw/excalidraw-mcp?ref=githubawesome.com)

No.20

![](https://githubawesome.com/content/images/2026/03/image-53.png)

The OpenClaw forks just keep coming! tinyclaw takes the core concepts of the Claw ecosystem and strips them down to the absolute bare minimum for educational purposes. If you want to learn exactly how these autonomous AI agents work under the hood without wading through hundreds of thousands of lines of code, this is the perfect starting point.

[https://github.com/TinyAGI/tinyclaw](https://github.com/TinyAGI/tinyclaw?ref=githubawesome.com)

No.21

![](https://githubawesome.com/content/images/2026/03/image-54.png)

This is hands down the most fun extension of the month. Pixel Agents is a free VS Code extension that turns your AI coding agents into animated pixel-art characters! Every time you open a Claude Code terminal, a little character spawns in a virtual office at the bottom of your screen. When the AI is writing code, the character types; when it searches, it reads a book. It is purely delightful.

[https://github.com/pablodelucca/pixel-agents](https://github.com/pablodelucca/pixel-agents?ref=githubawesome.com)

No.22

![](https://githubawesome.com/content/images/2026/03/image-55.png)

Anthropic literally gave a team of 16 Claude Opus instances $20,000 in API credits and told them to write a C compiler from scratch in Rust. The result is claudes-c-compiler—a 100,000-line, dependency-free C compiler that can actually build the Linux Kernel and DOOM! It sparked a massive debate on GitHub and Hacker News about the future of AI software engineering.

[https://github.com/anthropics/claudes-c-compiler](https://github.com/anthropics/claudes-c-compiler?ref=githubawesome.com)

No.23

![](https://githubawesome.com/content/images/2026/03/image-56.png)

If you are building AI agents, you need a fast terminal. Kaku is a deeply customized fork of WezTerm built specifically for AI coding on macOS. It is completely out-of-the-box, coming pre-configured with Starship prompts, smart directory jumping, syntax-highlighted git diffs, and intelligent auto-suggestions. It's an absolutely gorgeous, zero-config terminal replacement.

[https://github.com/tw93/Kaku](https://github.com/tw93/Kaku?ref=githubawesome.com)

No.24

![](https://githubawesome.com/content/images/2026/03/image-57.png)

Sometimes you just need a beautiful CLI tool. weathr is a stunning terminal UI for checking the forecast. It pulls real-time meteorological data and renders gorgeous ASCII-art weather icons, temperature graphs, and wind readouts right in your terminal. It is fast, lightweight, and makes your command line look incredibly aesthetic.

[https://github.com/Veirt/weathr](https://github.com/Veirt/weathr?ref=githubawesome.com)

No.25

![](https://githubawesome.com/content/images/2026/03/image-58.png)

tirith is a security tool you probably need if you are copying and pasting code from the internet. It's a Rust-based terminal gatekeeper that intercepts suspicious URLs, homograph attacks, and malicious ANSI injection payloads before they execute in your shell. It actively prevents those terrifying pipe-to-shell attacks that hackers use to hide malware in plain text.

[https://github.com/sheeki03/tirith](https://github.com/sheeki03/tirith?ref=githubawesome.com)

No.26

![](https://githubawesome.com/content/images/2026/03/image-59.png)

We featured Context Mode in a previous episode, but it blew up again in February. It solves the issue of massive log dumps ruining your AI’s context window. It isolates tool executions and uses fuzzy search to only feed relevant outputs back to the LLM. The newest update just made it incredibly stable and officially a must-have for anyone using local AI coding assistants.

[https://github.com/mksglu/claude-context-mode](https://github.com/mksglu/claude-context-mode?ref=githubawesome.com)

No.27

![](https://githubawesome.com/content/images/2026/03/image-61.png)

If you make YouTube videos or stream, Textream is a lifesaver. It is a free, native macOS teleprompter app that uses local, on-device AI speech recognition to highlight your script exactly as you speak it! It scrolls with you, pauses when you pause, and displays the text in a sleek Dynamic Island overlay right near your webcam so you never break eye contact.

[https://github.com/f/textream](https://github.com/f/textream?ref=githubawesome.com)

No.28

![](https://githubawesome.com/content/images/2026/03/image-62.png)

With the explosion of OpenClaw, everyone is asking: what do I actually do with it? awesome-openclaw-usecases is a curated GitHub repository tracking the most insane, brilliant, and profitable ways people are using OpenClaw agents to automate their lives, run headless businesses, and manage their local workflows. It is pure inspiration.

[https://github.com/hesamsheikh/awesome-openclaw-usecases](https://github.com/hesamsheikh/awesome-openclaw-usecases?ref=githubawesome.com)

No.29

![](https://githubawesome.com/content/images/2026/03/image-63.png)

Ars Contexta is a fascinating new take on personal knowledge management. It is an agent-native note-taking system. Instead of just writing markdown files, this tool is designed from the ground up to be readable and editable by both you and your AI agents natively. It acts as a shared brain where your agents can pull context, update facts, and organize your research continuously.

[https://github.com/agenticnotetaking/arscontexta](https://github.com/agenticnotetaking/arscontexta?ref=githubawesome.com)

No.30

![](https://githubawesome.com/content/images/2026/03/image-64.png)

Spacedrive has been building one of the best file managers on GitHub, and now they’ve open-sourced Spacebot. It integrates directly with their virtual filesystem architecture, meaning this AI assistant can semantically search, tag, and organize files across all of your hard drives and cloud storage accounts simultaneously. It's the ultimate file-management AI.

[https://github.com/spacedriveapp/spacebot](https://github.com/spacedriveapp/spacebot?ref=githubawesome.com)

No.31

![](https://githubawesome.com/content/images/2026/03/image-65.png)

RSS is making a comeback, but ClawFeed makes it autonomous. It’s a content feed reader managed entirely by your OpenClaw agent. Instead of you scrolling through hundreds of articles, your agent continuously reads your feeds in the background, categorizes them, summarizes the crucial points, and builds a daily intelligence briefing tailored exactly to your current projects.

[https://github.com/kevinho/clawfeed](https://github.com/kevinho/clawfeed?ref=githubawesome.com)

No.32

![](https://githubawesome.com/content/images/2026/03/image-66.png)

If you want your AI to feel a bit more present, companion by The Vibe Company is a lovely desktop app that puts a responsive AI avatar right on your screen. It watches your context, chats with you, and acts as a persistent co-pilot that breaks out of the traditional chat-box window to live organically on your desktop.

[https://github.com/The-Vibe-Company/companion](https://github.com/The-Vibe-Company/companion?ref=githubawesome.com)

No.33

![](https://githubawesome.com/content/images/2026/03/image-67.png)

And finally, at number 33, we have FastCode by HKUDS. This is an incredibly powerful open-agentic coding framework designed for rapid prototyping. You can feed it a research paper, and it will generate the corresponding code. Give it text, and it spins up a web UI or a backend server instantly. It’s an absolute powerhouse for taking ideas from concept to deployment at lightning speed.

[https://github.com/HKUDS/FastCode](https://github.com/HKUDS/FastCode?ref=githubawesome.com)
